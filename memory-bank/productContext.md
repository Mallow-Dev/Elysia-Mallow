# Product Context

Elysia Agent Management provides a standardized platform for defining, executing, and monitoring AI agent jobs. It supports flexible backend selection (Qdrant, Weaviate), robust health checks, and operational transparency.

## Overview

A unified system for agent orchestration, job specification, and backend management, enabling seamless integration and extensibility for AI workflows.

## Core Features

- Agent management and orchestration
- Codex Job Specification for standardized job execution
- Support for multiple vector database backends (Qdrant, Weaviate)
- Health checks and graceful fallback strategies
- Configuration via environment variables
- Progress tracking and audit logging

## Technical Stack

- Python (core platform)
- Docker (containerization)
- Qdrant, Weaviate (vector database backends)
- Markdown/JSON (specs and documentation)
